# normalize

Program w języku C, który normalizuje listę liczb do przedziału [0,1].

Dla podanych liczb program oblicza wartość minimalną i maksymalną,
a następnie dla każdej liczby x wykonuje operację:

(x - min) / (max - min)

Jeżeli użytkownik nie poda żadnych argumentów, program wyświetli krótką pomoc.
